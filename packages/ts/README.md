# @vatvit/freshen (TS / JS)

TypeScript/JavaScript implementation of **freshen** — a stale-while-revalidate
cache with stampede prevention.

```bash
npm install @vatvit/freshen
```

## Develop

Requires Node 20+.

```bash
npm install
npm test         # vitest
npm run build    # emit dist/ (esm + cjs + d.ts)
```

## Status

Boilerplate scaffold. Public API is not yet implemented — see the repository
requirements before adding code.
