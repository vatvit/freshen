# freshen (PHP)

PHP 8.4+ implementation of **freshen** — a stale-while-revalidate cache with
stampede prevention.

> This directory is the source of truth. It is subtree-split by CI into a
> read-only mirror repository that Packagist serves. Install via Packagist:

```bash
composer require vatvit/freshen
```

## Develop

Run in a container (PHP 8.4+):

```bash
composer install
composer test
```

## Status

Boilerplate scaffold. Public API is not yet implemented — see the repository
requirements before adding code.
